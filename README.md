# Search Application

This is a basic dictionary application which is built using the concept of string algorithms and hashing.

The key concepts used here are Trie and Hashing.

## Algorithms used:

1. Rabin Karp Algorithm
2. Trie Data Structure
3. Hashing
4. Z Algorithm
5. KMP map and Algo
